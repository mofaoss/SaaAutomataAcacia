# 前置设置

## 游戏设置

* 不论是窗口化还是全屏，游戏都必须是16:9。分辨率最好不低于1280×720

## SAA软件设置

* 核心设置：首次下载必须检查
* 软件启动时检查更新：建议开启
* 展示OCR结果：出现非预期行为可以打开看看是不是识别错别字了，如果存在错别字则去左侧`词表`上填上新的替换规则
* 展示运行时窗口截图：建议关闭
* 保存缩放比例：建议关闭
* 代理设置：填写梯子端口，没有则置空

##【计划功能】使用说明

* 生效起点：计划何时处于激活。例如起点周三，那么周一周二失效，周三到点开始激活计划
* 执行节点：到准确时间点自动执行，无需担心任何任务日程的时间冲突，任务会被智能排队执行
* 离线拉起：【自动登录】勾选自动打开游戏后，计划执行会帮你拉起游戏，你也可以设置结束后自动关闭游戏
* 助手挂机：设置可选隐藏托盘。计划任务的执行需要保持软件开启
